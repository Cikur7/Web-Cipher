# Web Cipher App

## Deskripsi
Aplikasi ini mengimplementasikan Vigenere, Playfair, dan Hill Cipher dalam C++ dengan antarmuka web.

## Cara Menjalankan
1. Pastikan Anda memiliki server C++ yang dapat menjalankan kode.
2. Buka `index.html` di browser Anda.
3. Masukkan plainteks, kunci (minimal 12 karakter), dan pilih jenis cipher.
4. Klik "Enkripsi" atau "Dekripsi" untuk mendapatkan hasil.
